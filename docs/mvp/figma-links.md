## 下一步建议

如果继续做 UI，建议顺序如下：

1. 在页面结构图中确认 MVP 页面边界与跳转关系。
2. 先把 `Writing Workbench` 和 `Sandbox Monitor` 细化成高保真设计稿。
3. 再统一组件尺寸、间距、按钮层级和状态样式。
4. 最后补 `角色库页`、`世界观页`、`审计中心页` 和 `版本历史页` 的线框。
